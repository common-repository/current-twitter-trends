== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload 'current-twitter-trends' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Goto Widgets and add this plug in in the sidebar and you are all set.
